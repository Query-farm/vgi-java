Worker-side Java implementation of the VGI (Vector Gateway Interface) protocol: serve catalogs, tables, and functions to Haybarn, Query Farm's DuckDB-derived engine, from an external Java process over Apache Arrow IPC. Your code registers functions and tables against a {@link farm.query.vgi.Worker} builder; the library handles the wire protocol, schema negotiation, batch streaming, pushdown, and transports.

Start with {@link farm.query.vgi.Worker}, register scalar functions by extending {@link farm.query.vgi.scalar.ScalarFn}, and see the project README for a runnable example worker.
